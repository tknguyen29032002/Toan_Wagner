# Product Overview

This is a Java implementation of the Clue (Cluedo) board game with a graphical user interface. The game features one human player competing against five computer players to solve a murder mystery by deducing the correct person, weapon, and room combination.

## Core Gameplay

- Players move around a custom board with 9 rooms, walkways, doorways, and secret passages
- Players make suggestions when entering rooms to gather clues
- Other players disprove suggestions by showing cards from their hand
- The game ends when a player makes a correct accusation or loses by making an incorrect one
- Computer players use AI logic to select moves, make suggestions, and determine when to accuse

## Key Features

- Turn-based gameplay with dice rolls determining movement range
- Visual board representation with player tokens and room layouts
- GUI panels showing game controls, known cards, and turn information
- Suggestion and accusation dialogs for human player interaction
- Computer AI that tracks seen cards and makes strategic decisions
