ANSWERS TO KEY QUESTIONS
================================================================================

Q1: What do you need to pass this class to populate the read-only "Current Room" field?
A1: We pass a Card object representing the current room. The dialog extracts 
    the room name using card.getName() and displays it in a non-editable 
    JTextField with white background to look read-only.

Q2: What do you need to pass to this class so it can show the options in the various combo boxes?
A2: We pass two List<String> parameters:
    - List<String> personNames: All person names from Board.getPersonNames()
    - List<String> weaponNames: All weapon names from Board.getWeaponNames()
    
    These populate the JComboBox options for Person and Weapon selections.

Q3: How will the main Clue Game know what selections were made in this box?
A3: We implemented two methods:
    - getSuggestion(): Returns a Solution object containing the selected 
      Person, Weapon, and Room cards (or null if cancelled)
    - wasSubmitted(): Returns boolean to distinguish between Submit vs Cancel
    
    The dialog is MODAL (blocks execution), so after setVisible(true), the
    main game can immediately call these methods to get the user's choice.

Class: SuggestionDialog extends JDialog

    - Centered on parent window

Return Values:
    - Solution getSuggestion(): Returns user's selection or null
    - boolean wasSubmitted(): Tracks if user clicked Submit vs Cancel

================================================================================
HOW TO USE IN MAIN GAME
================================================================================

Example Integration Code:

    // When human player enters a room
    Card currentRoomCard = new Card("Lounge", CardType.ROOM);
    
    // Create and show dialog (modal - blocks until user responds)
    SuggestionDialog dialog = new SuggestionDialog(
        this,  // main ClueGame frame
        currentRoomCard,
        board.getPersonNames(),
        board.getWeaponNames()
    );
    dialog.setVisible(true);
    
    // After dialog closes, process the suggestion
    if (dialog.wasSubmitted()) {
        Solution suggestion = dialog.getSuggestion();
        
        // Handle the suggestion through the board
        Card disproofCard = board.handleSuggestion(humanPlayer, suggestion);
        
        // Update GUI based on response
        if (disproofCard != null) {
            // Show disproof card to human player
            updateHumanSeenCard(disproofCard, cardOwner);
        } else {
            // No one could disprove
            controlPanel.setGuessResult("No one can disprove!");
        }
    }
